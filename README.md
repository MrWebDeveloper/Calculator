# Online Calculator
**A simple and responsive calculator powered by JavaScript.**

## Features
- multiplication
- addition
- division
- subtraction
- exponentiation
- percentage
- history *(saved in localStorage)*

## Technologies
- html
- css
- javascript
- local storage
